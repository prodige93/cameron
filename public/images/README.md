# Dossier Images

Placez vos images de réalisations dans ce dossier.

## Image requise

Pour la réalisation "Remplacement couverture et pose bardage Cedral", ajoutez l'image avec le nom :

**`bardage-cedral.jpg`**

Le chemin complet sera : `/images/bardage-cedral.jpg`

## Formats supportés

- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Note

Si l'image n'est pas trouvée, un gradient de couleur sera affiché automatiquement comme fallback.
